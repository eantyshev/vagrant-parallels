Vagrant Compatibility

When a new version of Vagrant is released, the Parallels provider is updated for it as soon as possible. You should always use the latest version of Vagrant and the latest version of the Parallels provider.

If for any reason you want to use an older version of Vagrant, you should always install a compatible version of the Parallels provider as described below.

Vagrant 1.4.x

The latest Parallels provider version compatible with Vagrant 1.4 is v0.2.2. If you are currently using Vagrant 1.4.x, you need to include the --plugin-version option in the installation command, specifying the Parallels provider version:

$ vagrant plugin install vagrant-parallels --plugin-version 0.2.2

Vagrant 1.3.x

The latest Parallels provider version compatible with Vagrant 1.3 is v0.0.9. When using Vagrant 1.3.x, the Parallels provider installation command should be:

$ vagrant plugin install vagrant-parallels --plugin-version 0.0.9
